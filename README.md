# 202242820 - Abdulaziz Altwaijri | Assignment 1 - Portfolio

A simple responsive personal portfolio website built using HTML, CSS, and JavaScript.

## Features
- About, Projects, Contact sections
- Responsive layout (mobile/tablet/desktop)
- Smooth scrolling navigation
- Light/Dark mode toggle (saved in localStorage)
- Contact form validation (no backend)

## Project Structure
- `index.html` - main page
- `css/styles.css` - styling
- `js/script.js` - interactivity
- `docs/ai-usage-report.md` - AI usage documentation
- `docs/technical-documentation.md` - technical details

## How to Run Locally
1. Download or clone this repository.
2. Open `index.html` in your browser.

I used AI tools to help with layout planning, responsive CSS ideas, JavaScript interactivity, and documentation writing. Details are in `docs/ai-usage-report.md`.
